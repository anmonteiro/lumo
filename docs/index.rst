Lumo
================

Lumo is a standalone ClojureScript environment that runs on
`Node.js <https://nodejs.org/en/>`_ and the V8 JavaScript engine.
It starts up instantaneously and has out-of-the-box access to the entire Node.js ecosystem.

Lumo is excels at creating fast starting command line utilities, equally with
long running servers.
